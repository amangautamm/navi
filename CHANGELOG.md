# Changelog

## [1.0.0] — 2025

### Initial Release

- Project Agent — scans up to 150 files, fixes errors across the entire codebase
- File Agent — works deeply on the current open file, applies changes directly to the editor
- Ghost text completions (Tab to accept)
- Auto-debug popup when errors are detected
- Left sidebar panel (always visible)
- Project / File scope toggle
- Seven mode tabs: Chat, Agent, Debug, Refactor, Build, Tests, Security
- One-click Fix Project and Fix File buttons
- Apply / Discard UI for reviewing AI changes before applying
- Session memory — save context that persists across sessions
- Custom coding rules — permanent AI instructions
- Three-model system: Primary, Coding, Fast
- Uncensored mode via dolphin-llama3
- Full keyboard shortcut support
- Right-click context menu
